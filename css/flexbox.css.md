# CSS: Flexbox

## Properties

- Parent (flex container)
  - `display`: `flex`
  - `flex-flow`
    - `flex-direction`
    - `flex-wrap`
  - `justify-content`
  - `align-items`
  - `align-content`
  - `gap`, `row-gap`, `column-gap`

- Children (flex items)
  - `order`
  - `flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]`
    - `flex-grow`
    - `flex-shrink`
    - `flex-basis`
  - `align-self`
