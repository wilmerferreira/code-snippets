# Docker: Compose

Compose is a tool for defining and running multi-container Docker applications.

- The ports use the following syntax `HOST:CONTAINER`

## [Commands](docker-commands.md#docker-compose)

Main commands:

- `docker-compose`
  - `docker-compose up`
  - `docker-compose down`
  - `docker-compose config`
